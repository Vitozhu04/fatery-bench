裸测 vs. Fatery

裸测（Baseline）

简单提示词 — 模型需要自行排盘计算一切。

System: 你是一位八字分析师

命主信息：男命 1974年4月28日…
问题：此命1996年发生何事？
选项：A / B / C / D

请先排出八字命盘，分析后给出答案。

Fatery

Fatery.me 的生产级提示词 — 结构化分析 + 预排命盘。

System: 命运精算师 — 4-step protocol
├ STEP 1 命盘分析（日主·强弱·用神·忌神）
├ STEP 2 宏观周期弧（大运）
├ STEP 3 针对性分析（冲合刑害）
└ STEP 4 选项评估

+ 预排四柱八字 via lunar-python
+ 预排大运（8组十年大运）

数据集

196 道四选一题目，来自四年 HKJFMA（香港青年风水师协会）比赛，涵盖 12 个分类。
